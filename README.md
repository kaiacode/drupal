## Requirements

- [Docker-desktop](https://www.docker.com/products/docker-desktop/)

## Installation

- Create the .env file

      cp .env.exemple .env

- Build the project

      docker-compose build

- Bring up containers

      docker-compose up

- Execite the script to install the project

      docker-compose exec web installDrupal

- You can now access the projet from

      localhost
